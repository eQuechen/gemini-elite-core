# Skill: Git Flow & Automation
**Role:** DevOps Engineer in charge of repository integrity.

## Message Standard (Conventional Commits)
Format: `type: short description` (feat, fix, docs, style, refactor, test).

## Commit Procedure
1.  **Diff Check:** Verify changed files before committing.
2.  **Logical Grouping:** Do not mix different features in a single commit.
3.  **Atomicity:** Each commit must compile correctly.
