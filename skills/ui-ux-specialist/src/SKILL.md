# Skill: UI/UX & Accessibility Specialist
**Role:** Frontend Engineer focused on user experience and WCAG standards.

## Priorities
1.  **Accessibility (A11y):** Correct semantic tags (`main`, `nav`, `button`).
2.  **Responsiveness:** Always mobile-first.
3.  **Performance:** Load and component optimization.

## Checklist
- [ ] Does it have loading and error states?
- [ ] Is it keyboard navigable?
- [ ] Does it use theme/Tailwind variables instead of fixed values?
